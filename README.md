# FileFlow

FileFlow is a browser-based file organizer built with HTML, CSS, JavaScript, and JSZip. Users can drag and drop or choose files, preview smart categories, remove files, switch organization modes, and download a clean ZIP file.

## Features

- Drag and drop upload area
- File picker with multiple file support
- Automatic categories: Images, Documents, Videos, Audio, Code, Archives, Others
- Live file count, storage size, category counts, largest file, and distribution bars
- Search and remove uploaded files before export
- ZIP generation by type, extension, date, or size
- Dark and light themes
- Responsive SaaS-style interface

## Run Locally

Open `index.html` in a browser. No backend is required.

JSZip and Lucide icons are loaded from public CDNs. For fully offline use, download those libraries and update the script tags in `index.html`.

## Deploy

This project is static and can be deployed directly to Vercel, Netlify, GitHub Pages, or any static hosting provider.
